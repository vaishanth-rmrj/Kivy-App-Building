# Kivy-App-Building
Using Kivy python package to create apps

## Installation of KIVY Material Design package:
1. Recommended to install python version 3.6
- For Ubuntu:

```
$ sudo apt update
$ sudo apt install python3.6
```

2. Installing KivyMD

```
pip install kivymd
```
